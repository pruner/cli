# `mocha` recipe

## Prerequisites
To use Pruner in your Mocha test project, you must install the `mocha` [NPM package](https://www.npmjs.com/package/mocha) globally.

`npm i mocha -g`

## Initializing for first-time use
`pruner init mocha`

# Troubleshooting
